{{ 'ADMIN.INSTRUCTIONS' | translate }}

Здесь вы можете управлять пользователями, ролями, правами и другими административными настройками. Выберите раздел ниже, чтобы увидеть краткие инструкции.

{{ 'MENU.USERS' | translate }}
  1. Просмотр списка пользователей и их свойств.
  2. Добавление нового пользователя: заполните форму, укажите роль и контактные данные.
  3. Редактирование: измените данные пользователя и сохраните.
  4. Удаление: выберите пользователя и подтвердите удаление.
 {{ 'MENU.ROLES' | translate }}
  1. Просмотрите доступные роли в системе.
  2. Создайте новую роль с нужными правами доступа.
  3. Изменяйте или удаляйте существующие роли по необходимости.
  4. Назначайте роли пользователям через раздел «Пользователи».
 {{ 'MENU.PERMISSIONS' | translate }}
  1. Определите наборы разрешений для различных действий.
  2. Группируйте разрешения по модулям и привязывайте к ролям.
  3. Проверяйте эффективные права пользователя через UI или логи.
 {{ 'MENU.NOTIFICATIONS' | translate }}
  1. Настройка типов уведомлений (email, в приложении и т.д.).
  2. Просмотр списка отправленных/ожидающих уведомлений.
  3. Редактирование шаблонов и расписания рассылок.
 {{ 'MENU.DEPARTMENTS' | translate }}
  1. Создавайте иерархию отделов/подразделений.
  2. Назначайте ответственных и связывайте отделы с проектами.
  3. Редактируйте информацию об отделе: название, описание, контакты.
 {{ 'MENU.SPECIALIZATIONS' | translate }}
  1. Добавление и управление специализациями сотрудников.
  2. Связывание специализаций с ролями и задачами.
  3. Импорт/экспорт каталога специализаций при необходимости.
